# password-manager-python
A simple password manager built with Python and Tkinter that generates secure passwords, stores credentials in JSON format, and allows users to search saved accounts.

# 🔐 Password Manager (Python)

A simple and user-friendly password manager built with Python and Tkinter.

## Features

- 🔑 Generate secure random passwords
- 💾 Save passwords locally in a JSON file
- 🔍 Search saved credentials
- 📋 Copy generated passwords to the clipboard (with pyperclip)
- 🖥️ Easy-to-use graphical interface

## Technologies Used

- Python 3
- Tkinter
- JSON
- Random
- Pyperclip (optional)

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/password-manager-python.git
   ```

2. Install the optional dependency:
   ```bash
   pip install pyperclip
   ```

3. Run the project:
   ```bash
   python password_manager.py
   ```
